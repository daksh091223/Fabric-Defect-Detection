Fabric Defect Detection 
------------------------------------------

This project presents a fabric defect detection system built using deep learning
techniques. The objective of the project is to automatically identify and classify
defects present in fabric images, along with recognizing normal fabric samples.

The system uses a convolutional neural network to learn visual features such as
texture variations, irregular patterns, and structural inconsistencies that
indicate the presence of defects. Based on these learned features, the model
predicts the category of defect present in the fabric image.

The project highlights how deep learning and computer vision can be applied to
solve real-world industrial problems in textile quality inspection. By automating
the defect detection process, the system helps reduce manual effort, improve
inspection consistency, and increase overall efficiency in fabric quality control.

The work also includes performance evaluation through accuracy and confusion
matrix analysis, which provides insights into how well different defect categories
are recognized by the model.
